Git basic use

Basic process

1: create server save files = git project (github,bitbucket...)
2: download git project to computer
3: edit,add,delete files
4: update files from git project (need update before upload)
5: upload files to server git project

Language by git

1: go to (github,bitbucket...) login create project
2: git clone
3: notepad edit file, or add file, delete file in fordel clone
4: git pull
5: git push (need git add, git commit)

Basic git

- git clone (download project of git to computer)
- git pull: (after clone/exit source have .git folder) load new download project of git to computer (file change)
- git push (after status,add,commit) upload file of computer to online project git
  - - need provide: account
- git status: view file in computer change when have git clone/pull (can old version of git)
- git add: add file/files change to git server
- git commit: add comment and ready push to server
